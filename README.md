---

## quickstart

In your first terminal window:

```bash

yarn install

```

```bash

yarn start

```

in a second terminal window:

```bash

yarn chain

```

Or alternatively:

```bash

yarn fork

```

in a third terminal window:

```bash

yarn deploy

```
